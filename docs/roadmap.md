# Roadmap

Free AI Audio Transcriber v1 is intentionally small: local transcription, preview, and TXT/SRT/VTT/LRC export.

Planned improvements:

- Better timeline segmentation.
- Improved SRT/VTT timestamp quality.
- Better Chinese and English punctuation processing.
- More complete multilingual transcription UX.
- Improved video file support.
- MP4 audio extraction in the browser.
- Batch transcription.
- PWA support.
- Model cache controls.
- Move model files to Cloudflare R2 for more predictable delivery.
- Optional cloud high-accuracy transcription.
- Optional advertising.
- More SEO pages.
- Chinese UI version.
- Dark mode.
- Drag-and-drop folder upload.

Non-goals for the MVP:

- User login.
- Paid subscription.
- Database.
- Server-side media upload.
- Paid transcription API.
- Long-term file storage.
- User history.
- Advertising scripts.
- Analytics scripts.
