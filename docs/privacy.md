# Privacy Notes

Free AI Audio Transcriber is designed as a privacy-friendly browser-local tool.

## What happens to files

- Files are processed locally in the user's browser by default.
- The MVP does not upload audio or video files to a server.
- The MVP does not store audio or video files.
- The MVP does not include user accounts, a database, or user history.

## Model downloads

The first transcription may download an AI model through Transformers.js. Browser caching may keep model files so future runs start faster.

## Future policy updates

If future versions add analytics, advertising, optional cloud transcription, or server-side storage, the privacy policy must be updated before those features are enabled.
